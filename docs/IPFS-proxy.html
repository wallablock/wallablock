post /wb/{dirCid}
Upload a new directory (wbDirCidPost)
The body shall be a form data with the name of the files, following the spec of a Wallablock IPFS directory.

Path parameters

dirCid (required)
Path Parameter — CID of the directory to be uploaded

Consumes

This API call consumes the following media types via the Content-Type request header:
  • multipart/form-data

Responses

201

The directory was accepted and created

400

The directory is not a valid Wallablock offer dir.

403

The requested CID is not in the blockchain and, therefore, shall not be stored

409

The produced CID does not match with the target CID
